# **App Name**: LearnSphere AI

## Core Features:

- Personalized Learning Path: Analyzes user's current knowledge and learning style to create a tailored ML learning path.
- AI Concept Explainer: Generates clear and concise explanations of complex ML concepts, adapting the tone and level of detail based on user feedback. Can use various external tools to determine what knowledge the user already possesses.
- Code Generation Tool: Generates Python code snippets for various ML tasks and algorithms based on user requests and context. Includes comments and explanations.
- Visual Learning Aids: Creates visualizations, such as graphs and diagrams, to illustrate ML concepts and data patterns. Dynamically adjusts the visuals based on user interaction and AI analysis.
- Audio Explanations: Provides audio explanations of ML concepts in different voices and tones to cater to auditory learners.
- Progress Tracking: Monitors user progress through the learning path, tracks completed lessons and provides feedback on areas for improvement.
- Adaptive Quizzes: Generates quizzes that adapt to the user's skill level.
- Flask Backend: Connects the application features with a Flask backend for data management and processing.

## Style Guidelines:

- Primary color: Deep sky blue (#42A5F5) for a sense of trust and innovation.
- Background color: Very light blue (#E3F2FD), almost white.
- Accent color: Royal blue (#1E88E5) for interactive elements.
- Body and headline font: 'Inter', a sans-serif font known for its clean and modern look.
- Use clear, consistent, and informative icons to represent different ML concepts and functionalities.
- Clean and intuitive layout with clear visual hierarchy to guide users through the learning path.
- Subtle animations and transitions to enhance user experience and provide visual feedback.
